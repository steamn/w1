По маршруту /promise/ ваш веб-ресурс должен возвращать в простом текстовом формате (не должно быть тегов, чистый текст кода на JavaScript) код функции task от одного аргумента x, т.е. function task(x){ return ... } которая возвращает промис, разрешённый в значение yes, если x < 18, и отвергающий с причиной no в противном случае.
